еҸ‘иЎЁдәҺ2021-05-17|жӣҙж–°дәҺ2021-05-17|еҚ•е…ғжөӢиҜ•
|йҳ…иҜ»йҮҸ:

unittest

unittest жҳҜ Python иҮӘеёҰзҡ„зұ» Junit еҚ•е…ғжөӢиҜ•жЎҶжһ¶гҖӮ

еғҸ Junit д№ӢдәҺ Java дёҖж ·пјҢunittest еҸҜз”ЁдәҺеҚ•е…ғжөӢиҜ•пјҢд№ҹеҸҜз”ЁдәҺ Web иҮӘеҠЁеҢ–жөӢиҜ•з”ҡиҮіжҺҘеҸЈжөӢиҜ•гҖӮunittest ж”ҜжҢҒжөӢиҜ•з”ЁдҫӢ/жөӢиҜ•з”ЁдҫӢйӣҶзҡ„жҹҘжүҫгҖҒз»„иЈ…пјҢиҝҳеҸҜд»ҘеңЁжөӢиҜ•з”ЁдҫӢ/жөӢиҜ•з”ЁдҫӢйӣҶеҶ…е…ұдә«ж•°жҚ®пјҢд№ҹж”ҜжҢҒж №жҚ®жқЎд»¶зӯӣйҖүжөӢиҜ•з”ЁдҫӢжү§иЎҢпјҢд»ҘеҸҠиҮӘеҠЁеҢ–з”ҹжҲҗжөӢиҜ•жҠҘе‘ҠгҖӮ

дҪҝз”Ё unittest еҸҜд»Ҙеҝ«йҖҹжҗӯе»әиҮӘеҠЁеҢ–жөӢиҜ•жЎҶжһ¶иҝӣиЎҢжөӢиҜ•гҖӮ

unittest ж ёеҝғз»„жҲҗ

Test Fixture

Test Fixture йҖҡеёёз”ЁжқҘеҒҡжөӢиҜ•з”ЁдҫӢзҡ„еҮҶеӨҮжҲ–иҖ…жё…зҗҶе·ҘдҪңгҖӮжҜ”еҰӮжөӢиҜ•ејҖе§ӢеүҚзҡ„ж•°жҚ®еҮҶеӨҮжҲ–иҖ…жөӢиҜ•з»“жқҹеҗҺзҡ„ж•°жҚ®жё…зҗҶзӯүгҖӮPython йҖҡиҝҮ setUp()гҖҒtearDown()гҖҒsetUpClass()гҖҒtearDownClass() иҝҷ 4 дёӘй’©еӯҗеҮҪж•°пјҲHookпјүжқҘе®һзҺ°жөӢиҜ•зҡ„еҮҶеӨҮе’Ңжё…зҗҶе·ҘдҪңгҖӮ

Test Case

Test Case жҳҜ unittest зҡ„жңҖе°ҸеҚ•е…ғпјҢдёҖдёӘ Test Case е°ұжҳҜдёҖдёӘжөӢиҜ•з”ЁдҫӢпјҢйҖҡеёё Test Case дјҡз»§жүҝ TestCase иҝҷдёӘеҹәзұ»гҖӮ

Test Suite

Test Suite жҳҜжөӢиҜ•еҘ—д»¶пјҢе°ұжҳҜжҲ‘д»¬еёёиҜҙзҡ„жөӢиҜ•з”ЁдҫӢйӣҶпјҢе®ғеҸҜд»ҘеҢ…еҗ«дёҖдёӘжҲ–еӨҡдёӘжөӢиҜ•з”ЁдҫӢгҖӮ

Test Loader

Test Loader з”ЁжқҘд»ҺжҸҗдҫӣзҡ„зұ»пјҲclassesпјүе’ҢжЁЎеқ—пјҲmodulesпјүдёӯз”ҹжҲҗжөӢиҜ•з”ЁдҫӢйӣҶпјҢй»ҳи®Өжғ…еҶөдёӢunittest дјҡжҸҗдҫӣдёҖдёӘ default test loaderгҖӮ

Test Runner

Test Runner жҳҜжөӢиҜ•жү§иЎҢеҷЁпјҢз”ЁжқҘиҝӣиЎҢжөӢиҜ•з”ЁдҫӢзҡ„жү§иЎҢе’ҢжөӢиҜ•з»“жһңзҡ„иҫ“еҮәгҖӮ

unittest иҝҗиЎҢеҺҹзҗҶ

зҹҘйҒ“дәҶ unittest зҡ„ 5 еӨ§ж ёеҝғзұ»пјҢжҲ‘д»¬зңӢдёӢ unittest зҡ„иҝҗиЎҢеҺҹзҗҶпјҢеҰӮеӣҫжүҖзӨәпјҡ

Test CasesеҢ…жӢ¬дёҖдёӘжҲ–иҖ…еӨҡдёӘ TestCase зұ»пјҢе…¶дёӯдҝқеӯҳдәҶе…·дҪ“зҡ„жөӢиҜ•иҝҮзЁӢпјҢдҪ еҸҜд»ҘеңЁжөӢиҜ•зұ»йҮҢдҪҝз”Ё Test FixtureпјҢдҫӢеҰӮsetUp()гҖҒtearDown() иҝӣиЎҢжөӢиҜ•ејҖе§ӢеүҚзҡ„еҮҶеӨҮе’Ңз»“жқҹеҗҺзҡ„жё…зҗҶе·ҘдҪңгҖӮ

TestSuiteеҢ…жӢ¬дёҖдёӘжҲ–иҖ…еӨҡдёӘ TestSuite зұ»пјҢе…¶дёӯ TestSuite еҢ…жӢ¬дәҶдёҖдёӘжҲ–еӨҡдёӘ TestCaseпјҢд№ҹеҸҜд»ҘеҢ…жӢ¬е…¶д»– TestSuiteгҖӮTestSuite йҖҡиҝҮ addTest() жҲ–иҖ… addTests() ж–№жі•жҠҠдёҖдёӘдёӘзҡ„жөӢиҜ•з”ЁдҫӢжҲ–иҖ…жөӢиҜ•з”ЁдҫӢйӣҶпјҲTestSuiteпјүз»„иЈ…иө·жқҘжҲҗдёәдёҖдёӘж–°зҡ„жөӢиҜ•з”ЁдҫӢйӣҶгҖӮ

TestLoaderзұ»еҠ иҪҪжң¬ең°жҲ–д»ҺеӨ–йғЁж–Үд»¶дёӯе®ҡд№үеҘҪзҡ„ TestCase жҲ–иҖ… TestSuitesгҖӮ

TestRunnerеҢ…жӢ¬TextTestRunnerзұ»пјҢ е®ғжҸҗдҫӣдәҶиҝҗиЎҢжөӢиҜ•зҡ„ж ҮеҮҶе№іеҸ°гҖӮжөӢиҜ•иҝҗиЎҢеҸҜд»ҘйҖҡиҝҮ unittest.main() жҲ–иҖ… python -m unittest xxx.py жқҘиҝҗиЎҢгҖӮ

Test Results CollectorеҢ…жӢ¬ TestResults зұ»пјҢе®ғдёәжөӢиҜ•з»“жһңжҸҗдҫӣдәҶдёҖдёӘж ҮеҮҶе®№еҷЁпјҢе®ғеӯҳеӮЁиҝҗиЎҢзҡ„жөӢиҜ•з”ЁдҫӢзҠ¶жҖҒпјҢдҫӢеҰӮ errorsгҖҒfailuresгҖҒskippedпјҢжөӢиҜ•зҡ„з»“жһңеҸҜд»ҘзӣҙжҺҘеңЁ Console иҫ“еҮәпјҢд№ҹеҸҜд»ҘдёәйҖҡиҝҮе…¶д»–еҪўејҸиҫ“еҮәпјҢдҫӢеҰӮ TextгҖҒresultгҖҒoutputгҖӮ

unittestзҡ„дҪҝз”Ё

unittestз®ҖеҚ•дҪҝз”Ё

е®ҡд№үдәҶдёҖдёӘжөӢиҜ•зұ» TestSampleпјҢе®ғз»§жүҝиҮӘ unittest.TestCse зұ»пјҢеҰӮжһңдҪҝз”Ё unittest жЎҶжһ¶пјҢжөӢиҜ•зұ»еҝ…йЎ»иҰҒз»§жүҝunittest.TestCse зұ»пјҢдё”жөӢиҜ•з”ЁдҫӢй»ҳи®Өд»Ҙ test ејҖеӨҙпјҲе®һйҷ…дёҠиҝҷдёӘеҸҜд»Ҙжӣҙж”№пјүгҖӮ

жөӢиҜ•з”ЁдҫӢжңү2дёӘпјҢеҲҶеҲ«дёә test_equal е’Ң test_not_equalгҖӮжіЁж„ҸжөӢиҜ•з”ЁдҫӢеңЁ unittest йҮҢзҡ„иЎЁзҺ°еҪўејҸжҳҜдёҖдёӘзұ»ж–№жі•гҖӮ

жү§иЎҢз»“жһңпјҡ

TestFixture зҡ„дҪҝз”Ё

еҰӮжһңжғіеңЁжөӢиҜ•з”ЁдҫӢжҲ–иҖ…жөӢиҜ•з”ЁдҫӢйӣҶејҖе§ӢеүҚпјҢжү§иЎҢжҹҗдәӣж“ҚдҪңпјҢ еңЁжөӢиҜ•з”ЁдҫӢжҲ–иҖ…жөӢиҜ•з”ЁдҫӢйӣҶз»“жқҹеҗҺеҶҚжү§иЎҢеҸҰеӨ–дёҖдәӣж“ҚдҪңпјҢйӮЈд№Ҳеә”иҜҘдҪҝз”Ё Test FixtureгҖӮ

TestFixtureеҢ…жӢ¬еҰӮдёӢ4дёӘж–№жі•

  • setUp()
    • setUp()ж–№жі•еңЁжҜҸдёҖдёӘжөӢиҜ•з”ЁдҫӢжү§иЎҢжөӢиҜ•еүҚйғҪдјҡжү§иЎҢгҖӮ
  • setUpClass()
    • setUpClass()ж–№жі•д»…еңЁж•ҙдёӘжөӢиҜ•зұ»ејҖе§Ӣжү§иЎҢеүҚжү§иЎҢ.setUpClass()ж–№жі•еҝ…йЎ»дҪҝз”Ё @classmethod жқҘиЈ…йҘ°гҖӮ
  • tearDown()
    • tearDown()ж–№жі•еңЁжҜҸдёҖдёӘжөӢиҜ•з”ЁдҫӢжү§иЎҢеҗҺйғҪдјҡжү§иЎҢгҖӮ
  • tearDownClass()
    • tearDownClass()ж–№жі•д»…еңЁж•ҙдёӘжөӢиҜ•зұ»з»“жқҹжү§иЎҢеҗҺжү§иЎҢ.tearDownClass()ж–№жі•еҝ…йЎ»дҪҝз”Ё @classmethod жқҘиЈ…йҘ°гҖӮ

setUp() е’Ң setUpClass() йҖҡеёёз”ЁжқҘиҝӣиЎҢжөӢиҜ•еүҚзҡ„еҮҶеӨҮе·ҘдҪңгҖӮдҫӢеҰӮпјҢи®ҝй—®ж•°жҚ®еә“иҺ·еҫ—жөӢиҜ•з”ЁдҫӢйңҖиҰҒзҡ„ж•°жҚ®зӯүгҖӮ

tearDown() е’Ң tearDownClass() йҖҡеёёз”ЁжқҘиҝӣиЎҢжөӢиҜ•еҗҺзҡ„жё…зҗҶе·ҘдҪңгҖӮдҫӢеҰӮпјҢжөӢиҜ•з»“жқҹеҗҺеҲ йҷӨжөӢиҜ•дә§з”ҹзҡ„ж•°жҚ®пјҢе°Ҷиў«жөӢиҜ•зі»з»ҹжҒўеӨҚиҮід№ӢеүҚзҡ„зҠ¶жҖҒзӯүгҖӮ

жү§иЎҢз»“жһңеҰӮдёӢпјҡ

иҝҗиЎҢжҢҮе®ҡж–Үд»¶еӨ№дёӢзҡ„жөӢиҜ•з”ЁдҫӢ

еңЁзңҹе®һе·ҘдҪңдёӯпјҢжҲ‘д»¬еёёеёёйңҖиҰҒд»…иҝҗиЎҢжҹҗдёҖдёӘжөӢиҜ•зұ»пјҢжҲ–иҖ…жҹҗдёҖдёӘж–Үд»¶еӨ№дёӢзҡ„жөӢиҜ•з”ЁдҫӢгҖӮжӯӨж—¶пјҢеҸҜд»ҘеҲ©з”Ё unittest зҡ„ main еҮҪж•°жқҘжҢҮе®ҡ module иҝҗиЎҢгҖӮ

unittest.main зҡ„иҜӯжі•пјҡ

е…¶еҗ„дёӘеҸӮж•°зҡ„еҗ«д№үеҰӮдёӢпјҡ

  • moduleпјҡжҢҮе®ҡеҫ…иҝҗиЎҢзҡ„ moduleпјҢй»ҳи®ӨжҳҜвҖңmainвҖқпјӣ
  • defaultTestпјҡеҚ•дёӘжөӢиҜ•зҡ„еҗҚеӯ—жҲ–иҖ…еӨҡдёӘжөӢиҜ•еҗҚеӯ—зҡ„з»„еҗҲпјҲеҝ…йЎ»иҰҒ iterableпјүпјӣ
  • argvпјҡдј йҖ’з»ҷзЁӢеәҸзҡ„дёҖз»„еҸҳйҮҸпјҢеҰӮжһңжІЎжңүжҢҮе®ҡпјҢйӮЈд№Ҳзі»з»ҹй»ҳи®ӨдҪҝз”Ё sys.argvпјӣ
  • testRunnerпјҡжҢҮе®ҡ unittest зҡ„ test runnerпјҢеҸҜд»ҘжҳҜ test runner зұ»жң¬иә«жҲ–иҖ… test runner зұ»е®һдҫӢгҖӮй»ҳи®Өжғ…еҶөдёӢпјҢmain еҮҪж•°дјҡи°ғз”Ё sys.exit()пјҢе№¶дё”дјҡеңЁеұҸе№•дёҠжҳҫзӨәжөӢиҜ•иҝҗиЎҢй”ҷиҜҜжҲ–иҖ…жҲҗеҠҹзҡ„жҸҗзӨәпјӣ
  • testLoaderпјҡеҝ…йЎ»жҳҜ TestLoader зұ»е®һдҫӢпјҢй»ҳи®ӨжҳҜdefaultTestLoaderпјӣ
  • exitпјҡй»ҳи®ӨжҳҜ TrueпјҢеҚіжөӢиҜ•иҝҗиЎҢе®Ңи°ғз”Ё sys.exit()пјҢеңЁдәӨдә’жЁЎејҸдёӢдҪҝз”Ёж—¶еҸҜжҢҮе®ҡдёә Falseпјӣ
  • verbosityпјҡз”ЁдәҺжҺ§еҲ¶жҳҫзӨәеңЁ console йҮҢзҡ„ log зӯүзә§пјҢжңү 0гҖҒ1гҖҒ 2 дёүз§ҚпјҢдёҖиҲ¬й»ҳи®Өдёәзӯүзә§ 1пјҢе…¶дёӯзӯүзә§ 2 жҳҫзӨәзҡ„ log жңҖиҜҰз»ҶгҖӮ

дёӢйқўжқҘзңӢдёҖдёӘ discover зҡ„дҫӢеӯҗ, еҒҮи®ҫжҲ‘д»¬зҡ„йЎ№зӣ®з»“жһ„еҰӮдёӢпјҡ

test_one.pyзҡ„еҶ…е®№еҰӮдёӢпјҡ

test_two.pyзҡ„еҶ…е®№еҰӮдёӢпјҡ

three.pyеҶ…е®№еҰӮдёӢпјҡ

main.pyзҡ„еҶ…е®№еҰӮдёӢпјҡ

жү§иЎҢз»“жһңеҰӮдёӢпјҡ

еҸҜд»ҘзңӢеҲ°пјҢos.path.join(os.path.dirname(-file-), вҖҳtestsвҖҷ) иҝҷдёӘе‘Ҫд»ӨиҺ·еҸ–дәҶ tests иҝҷдёӘж–Үд»¶еӨ№зҡ„и·Ҝеҫ„пјҢз„¶еҗҺжҲ‘йҖҡиҝҮ get_module_name_string иҝҷдёӘж–№жі•пјҢжҠҠ tests ж–Үд»¶еӨ№дёӢзҡ„жүҖжңү module зҡ„string иҺ·еҸ–еҮәжқҘпјҲж”ҫеҲ° mod_string_list дёӯеҺ»пјүпјҢжҺҘзқҖжҲ‘йҒҚеҺҶжҜҸдёҖдёӘиҺ·еҸ–зҡ„ module stringпјҢжҠҠе®ғеҜје…Ҙе№¶еҠ е…ҘеҲ° unittest зҡ„ suites дёӯеҺ»пјҢжңҖеҗҺжҲ‘жҢҮе®ҡдәҶ runner е№¶дё”иҝҗиЎҢгҖӮ

и§ӮеҜҹеҸ‘зҺ°пјҢеҸӘжңүtest_one.pyе’Ңtest_two.pyдёӯзҡ„жөӢиҜ•з”ЁдҫӢиў«жү§иЎҢдәҶпјҢиҖҢthree.pyдёӯзҡ„жөӢиҜ•з”ЁдҫӢжІЎжңүиў«жү§иЎҢгҖӮ

иҝҷжҳҜдёәд»Җд№Ҳе‘ўпјҹжіЁж„ҸеҸӮж•° TestLoader й»ҳи®Өжғ…еҶөдёӢпјҢTest Loader д»…д»…дјҡжҹҘжүҫжүҖжңүд»ҘвҖңtestвҖқејҖеӨҙзҡ„ .py ж–Үд»¶пјҢе№¶дё”еңЁиҝҗиЎҢжөӢиҜ•з”ЁдҫӢж—¶пјҢд»…дјҡй»ҳи®ӨиҝҗиЎҢд»ҘвҖңtestвҖқејҖеӨҙзҡ„жөӢиҜ•ж–№жі•пјҢеӣ дёә three.py жҳҜд»ҘвҖңthreeвҖқејҖеӨҙзҡ„е№¶дёҚжҳҜд»ҘвҖңtestвҖқејҖеӨҙзҡ„пјҢжүҖд»Ҙе®ғжІЎжңүжү§иЎҢгҖӮ

еҠЁжҖҒжҹҘжүҫжөӢиҜ•з”ЁдҫӢиҝҗиЎҢ

йҷӨеҺ»зӣҙжҺҘдҪҝз”Ё unittest.main ж–№ејҸеҠ иҪҪ module иҝҗиЎҢеӨ–пјҢunittest иҝҳж”ҜжҢҒйҖҡиҝҮ TestLoader дёӢзҡ„ discover ж–№жі•еҺ»жҹҘжүҫжөӢиҜ•з”ЁдҫӢгҖӮ

иҜӯжі•еҰӮдёӢпјҡ

unittest е…Ғи®ёдҪ д»ҺжҹҗдёӘж–Үд»¶еӨ№ејҖе§ӢпјҢйҖ’еҪ’жҹҘжүҫжүҖжңүз¬ҰеҗҲзӯӣйҖүжқЎд»¶зҡ„жөӢиҜ•з”ЁдҫӢпјҢе№¶дё”иҝ”еӣһдёҖдёӘеҢ…еҗ«иҝҷдәӣжөӢиҜ•з”ЁдҫӢзҡ„ TestSuite еҜ№иұЎпјҢunittest.TestLoader.discover ж”ҜжҢҒзҡ„еҸӮж•°еҰӮдёӢпјҡ

  • start_dirпјҡиө·е§Ӣж–Үд»¶еӨ№зҡ„и·Ҝеҫ„пјӣ
  • patternпјҲеҢ№й…ҚжЁЎејҸпјүпјҡй»ҳи®ӨжҗңзҙўжүҖжңүд»ҘвҖңtestвҖқејҖеӨҙзҡ„жөӢиҜ•ж–Үд»¶пјҢе№¶жҠҠиҝҷдәӣж–Үд»¶йҮҢзҡ„д»ҘвҖңtestвҖқејҖеӨҙзҡ„жөӢиҜ•з”ЁдҫӢжҢ‘йҖүеҮәжқҘпјӣ
  • top_level_dirпјҲж №зӣ®еҪ•пјүпјҡжөӢиҜ•жЁЎеқ—еҝ…йЎ»д»Һж №зӣ®еҪ•еҜје…ҘпјҢеҰӮжһң start_dir зҡ„дҪҚзҪ®дёҚжҳҜж №зӣ®еҪ•пјҢйӮЈд№Ҳеҝ…йЎ»жҳҫејҸжҢҮе®ҡ top_level_dirгҖӮ

дҫқж—§жҳҜд»ҘдёӢзӣ®еҪ•пјҡ

е…¶д»–ж–Үд»¶еҶ…е®№дёҚеҸҳпјҢжҠҠ main.py ж–Үд»¶з”Ё discover зҡ„ж–№ејҸж”№еҶҷеҰӮдёӢпјҡ

жү§иЎҢз»“жһңеҰӮдёӢпјҡ

иҝҗиЎҢеҗҺеҸ‘зҺ°з»“жһңи·ҹз”Ё unittest.main зҡ„ж–№ејҸдёҖиҮҙгҖӮ

жҢүйңҖз»„иЈ…жөӢиҜ•з”ЁдҫӢ

д»ҺдёҠйқўзҡ„дҫӢеӯҗзңӢеҲ°пјҢжүҖжңүд»ҘtestејҖеӨҙзҡ„pyж–Үд»¶дё”д»ҘtestејҖеӨҙзҡ„ж–№жі•йғҪдјҡиў«жү§иЎҢпјҢйӮЈдәӣеҸҲжІЎзҡ„д»ҘtestејҖеӨҙзҡ„жҲ‘д»¬еҰӮдҪ•жү§иЎҢе‘ўпјҹ

еңЁ unittest дёӯпјҢtestSuite зҡ„з»„иЈ…пјҢеҸҜд»Ҙз”ЁдёҠиҝ°зҡ„ж–№ејҸзӣҙжҺҘ discoverпјҢд№ҹеҸҜд»Ҙз”Ё unittest.TestSuite.addTest() ж–№ејҸжқҘж·»еҠ жөӢиҜ•з”ЁдҫӢеҲ° TestSuiteпјҢжҢҮе®ҡжөӢиҜ•з”ЁдҫӢжү§иЎҢгҖӮ

иҝҳжҳҜд»ҘдёҠдҫӢеӯҗпјҡ

е…¶д»–ж–Үд»¶еҶ…е®№дёҚеҸҳпјҢmain.pyжӣҙж”№жҲҗеҰӮдёӢпјҡ

жү§иЎҢз»“жһңеҰӮдёӢпјҡ

еңЁжң¬ж¬ЎжөӢиҜ•дёӯпјҢжҢ‘йҖүдәҶTestOneжөӢиҜ•зұ»дёӯзҡ„testAssertNotEqualж–№жі•гҖҒTestTwoжөӢиҜ•зұ»дёӯзҡ„equal_testж–№жі•е’ҢThreeзұ»дёӯзҡ„testAssertEqualж–№жі•пјҢе°Ҷд»–д»¬з»„иЈ…еҲ°дёҖдёӘTestSuiteйҮҢиҝҗиЎҢгҖӮ

йҖҡиҝҮ suit.addTest() зҡ„ж–№ејҸпјҢе°ұеҸҜд»ҘжҢүз…§йңҖиҰҒе®һзҺ°жҠҠдёҚеҗҢж–Үд»¶дёӢзҡ„жөӢиҜ•з”ЁдҫӢз»„иЈ…еҲ°еҗҢдёҖдёӘ suite жү§иЎҢзҡ„ж“ҚдҪңгҖӮ

з ҙйҷӨй»ҳи®Ө patternпјҢйҡҸеҝғжүҖж¬Іе‘ҪеҗҚжөӢиҜ•ж–Үд»¶

еңЁд»ҘдёҠзҡ„дҫӢеӯҗдёӯпјҢthree.pyж–Үд»¶дёӢзҡ„жөӢиҜ•з”ЁдҫӢйғҪжІЎжңүиў«жү§иЎҢпјҢе…¶еҺҹеӣ е°ұжҳҜ unittest жңүй»ҳи®Өзҡ„жҹҘжүҫ pattern еҰӮдёӢпјҡ

  • жҹҘжүҫжөӢиҜ•ж–Үд»¶пјҢй»ҳи®ӨжҹҘжүҫвҖңtest*.pyвҖқпјӣ
  • жҹҘжүҫжөӢиҜ•з”ЁдҫӢпјҢй»ҳи®ӨжҹҘжүҫвҖңtest*вҖқгҖӮ

жҲ‘д»¬еҸҜд»ҘйҖҡиҝҮжӣҙж”№жҹҘжүҫ pattern зҡ„ж–№ејҸжқҘжү§иЎҢжүҖжңүзҡ„жөӢиҜ•з”ЁдҫӢпјҢд»Қд»ҘдёҠиҝ°йЎ№зӣ®дёәдҫӢпјҡ

е…¶д»–ж–Үд»¶дёҚеҸҳпјҢжӣҙж”№ main.py дёәпјҡ

жү§иЎҢз»“жһңеҰӮдёӢпјҡ

жҲ‘д»¬жҠҠй»ҳи®Өзҡ„ pattern жӣҙж”№дёәвҖқ*.pyвҖңпјҢиҝҷж ·д»»дҪ•еңЁ tests ж–Үд»¶еӨ№дёӢзҡ„ py ж–Үд»¶йғҪеҸҜд»Ҙиў«жҹҘжүҫеҲ°гҖӮеҸҜд»ҘзңӢеҲ° three.py дёӢиҝҗиЎҢдәҶжөӢиҜ•з”ЁдҫӢtestAssertEqualпјҢдҪҶжҳҜtest_twoдёӢзҡ„вҖңequal_testвҖқиҝҷдёӘж–№жі•жІЎжңүиҝҗиЎҢпјҢйӮЈжҳҜеӣ дёәж–№жі•вҖңtestMethodPrefixвҖқеңЁиө·дҪңз”ЁгҖӮ

жҲ‘д»¬жқҘжӣҙж”№дёӢжөӢиҜ•ж–№жі•зҡ„й»ҳи®ӨжҹҘжүҫж–№ејҸпјҢ жӣҙж”№ main.py дёәеҰӮдёӢпјҡ

жү§иЎҢз»“жһңеҰӮдёӢпјҡ

жү§иЎҢзҡ„жөӢиҜ•з”ЁдҫӢеҸӘжңүequal_testпјҢеҸҜд»ҘеҸ‘зҺ°testMethodPrefixж”№еҸҳдәҶжҹҘжүҫжөӢиҜ•з”ЁдҫӢзҡ„й»ҳи®Өж–№ејҸгҖӮ

еҝҪз•ҘжөӢиҜ•з”ЁдҫӢжү§иЎҢ

unittest иҝҳж”ҜжҢҒеҝҪз•Ҙжү§иЎҢжҹҗдәӣжөӢиҜ•з”ЁдҫӢпјҢеҸӘиҰҒеңЁиҰҒеҝҪз•Ҙзҡ„жөӢиҜ•з”ЁдҫӢдёҠеҠ дёҠеҰӮдёӢиЈ…йҘ°еҷЁеҚіеҸҜпјҡ

  • @unittest.skip() жү§иЎҢж—¶зӣҙжҺҘеҝҪз•ҘжҺүиў«иЈ…йҘ°зҡ„жөӢиҜ•з”ЁдҫӢпјӣ
  • @unittest.skipIf() еҰӮжһң skipIf йҮҢзҡ„жқЎд»¶жҲҗз«ӢпјҢжү§иЎҢж—¶зӣҙжҺҘеҝҪз•ҘжҺүиў«иЈ…йҘ°зҡ„жөӢиҜ•з”ЁдҫӢпјӣ
  • @unittest.skipUnless() ж°ёд№…еңЁжү§иЎҢж—¶еҝҪз•Ҙиў«иЈ…йҘ°зҡ„жөӢиҜ•з”ЁдҫӢпјҢйҷӨйқһ skipUnless йҮҢзҡ„жқЎд»¶жҲҗз«Ӣпјӣ
  • @unittest.expectedFailureжңҹжңӣиў«иЈ…йҘ°зҡ„жөӢиҜ•з”ЁдҫӢжҳҜеӨұиҙҘзҡ„пјҢеҰӮжһңжҳҜеӨұиҙҘзҡ„пјҢеҲҷжӯӨжқЎжөӢиҜ•з”ЁдҫӢе°Ҷиў«ж Үи®°дёәжөӢиҜ•йҖҡиҝҮгҖӮ

дёӢйқўжқҘйҖҡиҝҮдёҖз»„жөӢиҜ•жқҘжҳҫзӨәеҰӮдҪ•еҝҪз•ҘжөӢиҜ•з”ЁдҫӢжү§иЎҢпјҡ

жү§иЎҢз»“жһңпјҡ

unittestжЎҶжһ¶еҲӣе»әжөӢиҜ•зҡ„жӯҘйӘӨ

  • зј–еҶҷдёҖдёӘжөӢиҜ•зұ»пјҢиҝҷдёӘжөӢиҜ•зұ»еҝ…йЎ»з»§жүҝ TestCase иҝҷдёӘеҹәзұ»пјҢ жөӢиҜ•зұ»жүҖеҜ№еә”зҡ„ .py ж–Үд»¶й»ҳи®ӨиҰҒд»Ҙ test ејҖеӨҙпјӣ
  • еңЁиҝҷдёӘжөӢиҜ•зұ»дёӢйқўеҶҷдҪ зҡ„жөӢиҜ•ж–№жі•пјҢжҜҸдёӘжөӢиҜ•ж–№жі•еә”иҜҘеҢ…жӢ¬дёҖдёӘжөӢиҜ•зҡ„е®Ңж•ҙжӯҘйӘӨпјҢжөӢиҜ•ж–№жі•иҰҒй»ҳи®Өд»Ҙ test ејҖеӨҙпјӣ
  • йҖҡиҝҮ unittest.main()гҖҒrunner.run() жҲ–иҖ… python -m зҡ„ж–№ејҸжқҘи°ғз”ЁиҝҷдәӣжөӢиҜ•з”ЁдҫӢгҖӮ
