  • дҪҝз”ЁJavaжіӣеһӢзұ»пјҢж”ҜжҢҒеӨҡж•°жҚ®зұ»еһӢзҡ„ж•°з»„еӯҳеӮЁ
  • еҠЁжҖҒж•°з»„пјҢжҳҜжҢҮж•°з»„зҡ„е®№йҮҸеҠЁжҖҒдјёзј©пјҢж•°з»„еңЁе®ҡд№үзҡ„ж—¶еҖҷжҳҜеҲқе§ӢеҢ–е®№йҮҸпјҢеҰӮжһңеўһеҠ зҡ„ж•°з»„е…ғзҙ зҡ„дёӘж•°и¶…иҝҮзӣ®еүҚзҡ„ж•°з»„е®№йҮҸпјҢе°ұдјҡж–°з”ҹжҲҗдёҖдёӘж–°зҡ„ж•°з»„пјҲжҳҜеҪ“еүҚж•°з»„зҡ„2еҖҚзҡ„е®№йҮҸпјүпјҢе°ҶеҺҹжңүж•°з»„зҡ„е…ғзҙ еӨҚеҲ¶еҲ°ж–°зҡ„ж•°з»„дёӯпјҢе№¶е°ҶеҺҹж•°з»„жҢҮеҗ‘жҖ§еҲ°ж–°ж•°з»„пјҢеҺҹж•°з»„дёәз©әеҲҷдјҡиў«Javaеһғеңҫеӣһж”¶жңәеҲ¶еӣһж”¶

з®ҖеҚ•еҲҶжһҗеҠЁжҖҒж•°з»„зҡ„ж—¶й—ҙеӨҚжқӮеәҰ

  • ж·»еҠ е…ғзҙ  вҖ“ O(n) (иҖғиҷ‘жңҖеқҸзҡ„жғ…еҶө)
    • addLast(e) вҖ“ O(1)
    • addFirst(e) вҖ“ O(n)
    • add(index, e) вҖ“ дёҘж ји®Ўз®—йңҖиҰҒдёҖдәӣжҰӮзҺҮзҹҘиҜҶ O(n/2) = O(n)
    • resize вҖ“ O(n)
    • еҰӮжһңеҸӘеҜ№жңҖеҗҺдёҖдёӘе…ғзҙ ж“ҚдҪңеә”жҳҜO(1)пјҢеӣ дёәеј•е…ҘдәҶresizeпјҢresizeжҳҜO(n)зә§еҲ«зҡ„пјҢиҖғиҷ‘жңҖеқҸзҡ„жғ…еҶөпјҢеҲҷжҲҗдёәO(n)
  • еҲ йҷӨж“ҚдҪң вҖ“ O(n) (иҖғиҷ‘жңҖеқҸзҡ„жғ…еҶө)
    • removeLast(e) вҖ“ O(1)
    • removeFirst вҖ“ O(n)
    • remove(index, e) вҖ“ O(n/2) = O(n)
    • еҰӮжһңеҸӘеҜ№жңҖеҗҺдёҖдёӘе…ғзҙ ж“ҚдҪңеә”жҳҜO(1)пјҢеӣ дёәеј•е…ҘдәҶresizeпјҢresizeжҳҜO(n)зә§еҲ«зҡ„пјҢиҖғиҷ‘жңҖеқҸзҡ„жғ…еҶөпјҢеҲҷжҲҗдёәO(n)
  • дҝ®ж”№ж“ҚдҪңпјҡе·ІзҹҘзҙўеј•O(1)пјӣжңӘзҹҘзҙўеј•O(n)
    • set(index, e) вҖ“ O(1)
  • жҹҘиҜўж“ҚдҪң: е·ІзҹҘзҙўеј•O(1)пјӣжңӘзҹҘзҙўеј•O(n)
    • get(index) вҖ“ O(1)
    • contains вҖ“ O(n)
    • find(e) вҖ“ O(n)
  • resize O(n)зҡ„еӨҚжқӮеәҰеҲҶжһҗ
    • еҒҮи®ҫеҪ“еүҚcapacity=8пјҢе№¶дё”жҜҸдёҖж¬Ўж·»еҠ ж“ҚдҪңйғҪдҪҝз”ЁaddLast
    • з¬¬9ж¬ЎaddLastж“ҚдҪңпјҢи§ҰеҸ‘resizeпјҢжҖ»е…ұиҝӣиЎҢдәҶ17ж¬Ўж“ҚдҪң
    • е№іеқҮпјҢжҜҸж¬ЎaddLastж“ҚдҪңпјҢиҝӣиЎҢ2ж¬Ўеҹәжң¬ж“ҚдҪң
    • з”ұжӯӨжҖ»з»“пјҡ
      • еҒҮи®ҫcapacity=nпјҢn+1ж¬ЎaddLastпјҢи§ҰеҸ‘resizeпјҢжҖ»е…ұиҝӣиЎҢ2n+1ж¬Ўеҹәжң¬ж“ҚдҪң
      • е№іеқҮпјҢжҜҸж¬ЎaddLastж“ҚдҪңпјҢиҝӣиЎҢ2ж¬Ўеҹәжң¬ж“ҚдҪң
      • иҝҷж ·еқҮж‘Ҡи®Ўз®—пјҢж—¶й—ҙеӨҚжқӮеәҰжҳҜO(1)зҡ„
      • еңЁиҝҷдёӘдҫӢеӯҗйҮҢпјҢиҝҷж ·еқҮж‘Ҡи®Ўз®—пјҢжҜ”и®Ўз®—жңҖеқҸжғ…еҶөеҸҲж„Ҹд№ү

еқҮж‘ҠеӨҚжқӮеәҰе’ҢйҳІжӯўеӨҚжқӮеәҰзҡ„йңҮиҚЎ

  • resizeзҡ„еӨҚжқӮеәҰеҲҶжһҗ

    • еҒҮи®ҫеҪ“еүҚcapacity=8пјҢе№¶дё”жҜҸдёҖж¬Ўж·»еҠ ж“ҚдҪңйғҪдҪҝз”ЁaddLastпјҢйңҖиҰҒз»ҸиҝҮ8жӯӨзҡ„ж·»еҠ пјҢжҜҸдёҖж¬Ўзҡ„еӨҚжқӮеәҰйғҪжҳҜO(1)
    • жҺҘдёӢжқҘж·»еҠ з¬¬9дёӘе…ғзҙ пјҢеҲҷдјҡи§ҰеҸ‘resizeпјҢжҖ»е…ұиҝӣиЎҢдәҶ17жӯӨзҡ„еҹәжң¬ж“ҚдҪңпјҢ9жӯӨзҡ„ж·»еҠ ж“ҚдҪң + 8жӯӨзҡ„иҪ¬з§»ж“ҚдҪң
    • еҜ№дәҺ9ж¬Ўж“ҚдҪңпјҢе№іеқҮпјҢжҜҸж¬ЎaddLastж“ҚдҪңпјҢжҳҜиҝӣиЎҢ2ж¬Ўеҹәжң¬ж“ҚдҪңпјҲ17/9вүҲ2пјү
    • ж №жҚ®д»ҘдёҠз»“и®әпјҢеҒҮи®ҫcapacity=nпјҢn+1жӯӨaddLastпјҢи§ҰеҸ‘resizeпјҢжҖ»е…ұиҝ‘дәҶ2n+1ж¬Ўеҹәжң¬ж“ҚдҪңпјҢе№іеқҮпјҢжҜҸж¬ЎaddLastж“ҚдҪңпјҢиҝӣиЎҢ2ж¬Ўеҹәжң¬ж“ҚдҪңпјҢиҝҷж ·еқҮж‘Ҡи®Ўз®—пјҢж—¶й—ҙеӨҚжқӮеәҰжҳҜO(1)зҡ„пјҒ
    • еңЁиҝҷдёӘдҫӢеӯҗдёӯпјҢиҝҷж ·еқҮж‘Ҡи®Ўз®—пјҢжҜ”и®Ўз®—жңҖеқҸжғ…еҶөжңүж„Ҹд№ү

  • еқҮж‘ҠеӨҚжқӮеәҰ amortized time complexity

    • йҖҡиҝҮдёҠйқўзҡ„д»Ӣз»ҚпјҢзҹҘйҒ“addLastзҡ„еқҮж‘ҠеӨҚжқӮеәҰжҳҜO(1)зә§еҲ«пјҢеҗҢзҗҶremoveLastзҡ„еқҮж‘ҠеӨҚжқӮеәҰд№ҹжҳҜO(1)зә§еҲ«
