Appium Page Object模式

PO模式封装的主要组成元素

  • PageObject对象：完成对页面的封装
  • Driver对象：完成对web、android、ios、接口的驱动
  • 测试用例：调用Page对象实现业务并断言
  • 数据封装：配置文件和数据驱动
  • Utils：其他功能封装，改进原生框架不足
