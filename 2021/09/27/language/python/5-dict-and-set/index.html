集合

集合是一种不允许元素出现重复的容器。

创建

与字典（dict）类似，集合（set）也是由一对花括号（{}）创建。但是，容器内的元素不是键值对。

示意图如下：

同字典类似，集合内的元素必须是可哈希类型（hashable）。

这就意味着 list、dict 等不可哈希的对象不能作为集合的元素。

另一种创建集合的方法，是通过 Python 的内置的 set 函数，参数类型为可迭代对象 Iterable。
