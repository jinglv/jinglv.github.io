Docker的入门应用

总结：

  1. docker run 镜像名，根据run后面的镜像运行一个容器
  2. 在运行之前在本地仓库中查找对应的镜像
    • 找到直接使用
    • 找不到，则会到远程仓库下载

镜像的获取

  • pull from registry(online)从registry拉取
    • public(公有)
    • private(私有)
  • Build from Dockerfile(online)从Dockerfile构建
  • Load from file(online)文件导入（离线）
